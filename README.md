## 📌 Overview
The **MRDU Campus Lost & Found Portal** is a real-time web application designed to help university students easily report and recover lost items on campus. Built with a modern, responsive Glassmorphism UI and powered by a cloud database, this platform bridges the gap between students who have lost valuables and those who have found them.

### 🚀 Live Demo
**Experience the live portal here:** [https://manojeragoni2006-alt.github.io/rtp/]

---

## ✨ Key Features
* **Real-Time Database Integration:** Powered by Google Firebase (Cloud Firestore) to ensure lost and found items update instantly across all devices without needing to refresh the page.
* **Modern UI/UX Design:** Features a dynamic animated gradient background, Apple-style frosted glass panels (Glassmorphism), and magnetic hover animations.
* **Authentication System:** Includes a prototype registration and login gateway to ensure only campus students and admins can access the portal.
* **Smart Search:** A built-in filtering system allows students to instantly search the database for specific lost or found items.
* **Responsive Layout:** Built with CSS Flexbox to seamlessly transition from a side-by-side desktop dashboard to a stacked mobile-friendly view.
